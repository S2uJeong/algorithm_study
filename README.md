모여요! 알고리즘 스터디

## 스터디 규칙
1. 과제 : 일주일에 같은 문제 6개를 각자 풀이한다.
2. 스터디 : 랜덤 2문제를 풀이한다.
3. 스터디 시간에는 과제/스터디 각 1문제씩 풀이를 발표한다.
    
## Git 규칙
- ISSUE : 화요일마다 문제를 선정하고 ISSUE로 등록
  ```
  Summary
  - Date	
  - Difficulty	
  - Link	
  Insight
  ```
- 디렉토리 구조 : {문제사이트명}/{사이트약자_문제번호}/
- 파일명 규칙 : {영문명}.py
- Commit
  - ISSUE 번호를 꼭! 넣어서 commit 
  - 새로 생성인지 `[feat]`, 고친건지 명시 `[refac]`
  - 예시) [feat] #6 최수정
